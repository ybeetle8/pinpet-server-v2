## 项目概述
pinpet Server 是一个基于 Rust 开发的服务端应用程序，主要用于监听和处理 Solana 区块链上的 pinpet 合约事件，并提供相关的 API 服务。该项目整合了 Solana 区块链数据，使用 RocksDB 进行本地数据存储，并通过 RESTful API 为前端应用提供数据访问接口。
